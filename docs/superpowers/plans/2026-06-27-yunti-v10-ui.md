# Yunti V10 UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a calmer V10 iOS quiz experience with a browser-reviewed visual prototype, smoother auto-next behavior, cleaner bottom navigation, and safer question parsing.

**Architecture:** Keep the app in native UIKit. Use a temporary browser prototype to validate layout and motion direction before applying changes to `ViewController.swift`. Keep parser logic isolated in `QuestionParser.swift` and verify it with a local script.

**Tech Stack:** Swift/UIKit, PDFKit, UniformTypeIdentifiers, Xcode project files, Codemagic YAML, Python verification scripts, local HTML/CSS prototype.
